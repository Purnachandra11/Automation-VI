📋 Important Notes

Before Starting Tests, Please Ensure the Following Steps
  • Click the "Download Sample File" button, fill in the required test data, and start the test.
  • Start both ADB and Appium servers.
  • Enable USB Debugging on your device:
    1. Open Settings → About Phone
    2. Tap "Build Number" 7 times to enable Developer Options
    3. Go back to Settings → Developer Options
    4. Enable "USB Debugging"
  • For USB connection:
    • Enable "Install via USB"
  • For Wireless connection:
    • Enable "Wireless Debugging"
    • Note the following details:
      • IP Address (e.g., 100.84.166.5)
      • Port (e.g., 38819)
      • Pairing Code (e.g., 499861)
  • Connect your mobile device via USB or Wireless connection.
  • Ensure that both your device and computer are connected to the same VPN (for wireless connection).
